# ai-integration

This repository contains examples for working with prompt engineering and OpenAI models.

- `chatgpt-prompt-engineering-for-developers/l1_prompting_guidelines.ipynb`: basic prompt engineering guidelines and tactics.
- `chatgpt-prompt-engineering-for-developers/l2_iterative_prompting.ipynb`: iterative prompt development examples inspired by the course lesson on iterative prompting.
- `chatgpt-prompt-engineering-for-developers/INTRODUCTION.md`: documentation of the course introduction and key concepts for prompt engineering.
